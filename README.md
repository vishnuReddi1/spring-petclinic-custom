# Spring PetClinic Custom

A customized version of the **Spring PetClinic** application, created for learning, experimentation, and extending the original project with additional features.

> **Note:** This project is based on the original Spring PetClinic maintained by the Spring team. The original project and its license are respected.

---

## About the Project

Spring PetClinic is a sample Spring Boot application that demonstrates best practices for building enterprise Java applications.

This repository is my customized version, where I am experimenting with new features, improving the UI, and learning modern Spring Boot development.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/vishnuReddi1/spring-petclinic-custom.git
```

Move into the project:

```bash
cd spring-petclinic-custom
```

Run the application:

```bash
./mvnw spring-boot:run
```

The application will be available at:

```
http://localhost:8080
```
